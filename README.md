
O "Mecânica Fácil" é um sistema de gestão desktop, desenvolvido em Java e com banco de dados 
MySQL, projetado para auxiliar pequenas oficinas mecânicas no controle de suas operações diárias. 
O objetivo principal é centralizar e organizar informações de clientes, veículos e serviços, 
substituindo anotações manuais e planilhas, para otimizar o tempo do mecânico e melhorar o 
atendimento.
